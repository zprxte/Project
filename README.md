🚗 Barrier Gate & Parking Management System
ระบบบริหารจัดการลานจอดรถและไม้กั้นอัจฉริยะ (Web Application) พัฒนาด้วย PHP และ MySQL ระบบถูกออกแบบมาเพื่ออำนวยความสะดวกตั้งแต่การลงทะเบียนผู้ใช้, การบันทึกข้อมูลรถยนต์, การจัดการการเข้า-ออก, ระบบชำระเงิน ไปจนถึงหน้าต่างควบคุมสำหรับผู้ดูแลระบบและผู้บริหาร
🌟 ฟีเจอร์เด่น (Key Features)
ระบบได้แบ่งระดับผู้ใช้งาน (User Roles) ออกเป็น 3 ส่วน เพื่อการจัดการที่มีประสิทธิภาพ:
👤 1. ผู้ใช้งานทั่วไป (User)
• ระบบสมาชิก: ลงทะเบียนและเข้าสู่ระบบได้อย่างปลอดภัย
• จัดการข้อมูลรถ: เพิ่ม แก้ไข ข้อมูลรถยนต์ส่วนตัว
• บันทึกการจอด: บันทึกเวลาเข้า-ออกลานจอดรถ
• ประวัติการใช้งาน: ตรวจสอบประวัติการจอดรถย้อนหลัง
• ชำระเงินออนไลน์: แจ้งชำระเงินค่าบริการจอดรถได้อย่างสะดวก
🛡️ 2. ผู้ดูแลระบบ (Admin)
• Admin Dashboard: แผงควบคุมหลักสำหรับจัดการข้อมูลทั้งหมด
• ตรวจสอบข้อมูล: ดูข้อมูลการจอดรถและสมาชิกในระบบ
• จัดการการชำระเงิน: ตรวจสอบและยืนยันสลิปการโอนเงิน
• จัดการสิทธิ์: สามารถพิจารณาและเพิกถอนสิทธิ์การใช้งาน
📊 3. ผู้บริหาร (Executive)
• Executive Dashboard: ดูภาพรวมการทำงานของระบบ สถิติรายได้ และข้อมูลสำคัญเพื่อการตัดสินใจ
📸 หน้าจอการใช้งาน (Screenshots)

หน้าจอเข้าสู่ระบบและลงทะเบียน (Login & Registration)
